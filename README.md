# Shah Alam Local 30

A research-backed directory of 30 Shah Alam businesses with no discoverable independent website, plus a tailored concept website for each business.

The 30 concepts use 30 named visual identities and 30 business-specific hero compositions rather than a shared landing-page template. Every concept includes:

- an embedded, address-pinned Google Map and an external Google Maps link
- click-to-call phone actions
- a persistent WhatsApp quick action and industry-specific enquiry form
- project-generated concept photography, clearly labelled for replacement with owner-approved business photos

Generated concept images live in `assets/images/`. They are not photographs of the named businesses and must not be presented as such.

## Run

```bash
npm start
```

Open `http://localhost:4173`. Each concept has its own stable query URL, for example `site.html?id=restoran-hatinie`.

## Research standard

Checked 15 July 2026. “No website” means no independent business-owned domain was discoverable through a business-name + Shah Alam search. Facebook, Instagram, Waze, Google Maps, delivery platforms and directory profiles are treated as third-party presences, not independent websites. This is indicative research and must be rechecked before outreach or publication.

All concept sites are marked unofficial. Public contact details are included only to demonstrate useful calls to action and must be confirmed with each owner.
